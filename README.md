## ğŸ¯ About The Project

This project focuses on an **Exploratory Data Analysis (EDA)** of urban trees in Paris, based on an open dataset provided by the City of Paris.

The main objectives are to:

- Analyze the distribution of trees across Paris districts
- Study species diversity and tree characteristics
- Identify patterns related to tree condition and maintenance
- Provide data-driven insights to support urban planning and green space management

The analysis is conducted in a **Jupyter Notebook**, combining data cleaning, visualization, and interpretation.

---

## ğŸ—ƒï¸ Dataset

The dataset comes from **Paris Open Data** and contains detailed information about urban trees, including:

- Geographic location
- Species and genus
- Physical characteristics (height, diameter)
- Condition and maintenance indicators

This dataset enables both **descriptive statistics** and **visual exploration** of Parisâ€™ urban forest.

## âš”ï¸ Getting Started

This project runs locally using **Python** and **Jupyter Notebook**.

### Prerequisites

- Python 3.x
- pip
- Jupyter Notebook

### Installation

1. Clone the repository:

       git clone https://github.com/joelle-jnbaptiste/SchoolProject---Paris-trees-analysis.git

2. Install dependencies:

       pip install pandas matplotlib seaborn jupyter

3. Launch Jupyter Notebook:

       jupyter notebook
